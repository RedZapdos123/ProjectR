About the website and model:

This model makes use of the following libraries:

  • Flask
  • SciKit Learn
  • NumPy
  • Pandas
  • eXtreme Gradient Booster
  • Oputna
  • Warnings

The daset used is Kaggle Fish Market dataset.

The website is made using HTML5, CSS and Javascript, with responsive web design for most screen sizes, for the deployment of the model.

Warning: This model as of now makes use of a small dataset, so it can only classify between: Perch, Bream, Roach, Pike, Smelt, Parkki and Whitefish.
